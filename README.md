## 🏗️ VProfile Multi-VM DevOps Infrastructure

A production-like multi-tier Java application infrastructure provisioned using **Vagrant, VirtualBox, and Linux**.

The project consists of five virtual machines:

```text
Client
   │
   ▼
Nginx Reverse Proxy
(web01)
   │
   ▼
Apache Tomcat
(app01)
   │
   ├──► MariaDB
   │    (db01)
   │
   ├──► Memcached
   │    (mc01)
   │
   └──► RabbitMQ
        (rmq01)
```
